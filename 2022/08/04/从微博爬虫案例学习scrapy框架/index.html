Spider(爬虫)

分析域名结构

时间范围：https://s.weibo.com/weibo?q=%23实现祖国统一的决心坚如磐石%23&timescope=custom:2022-08-01:2022-08-04

爬取实现

class scrapy.spider.Spider，所有爬虫都必须继承自此类。

爬虫property

  1. spider类 的 basic properties

    • name

      爬虫的名字，必须唯一，通常以二级域名命名。

    • allowed_domains

      爬虫允许爬取的域名列表。

    • settings

      获得Settings的实例，使用其中的全局变量。

    其它property

    • start_urls

      无特定域名要求时，被爬取的页面域名列表。

  2. url参数

    可在settings中进行微博高级搜索的筛选条件配置，部分由util中的函数转换为url中的传参形式。

  3. 获取搜索关键词

    可在settings 或 文件中配置搜索的关键词列表，并对超话类关键词中的"#"编码。

  4. 一些关于搜索阈值 和 数据库的其它属性初始值

爬虫method

spider类中的basic methods

  • start_requests()

    解析得到请求 requests，并交由 Downloader 发送，接受响应 response 交由回调函数处理。通常使用 Requests 对象实现。

    class scrapy.http.Request

    通常通过 Spider 生成 Request 对象，通过 Donwloader 执行，获取response 。此类中含有其它功能性子类。

    参数：

    • url：发送 request 的域名
    • callback(default=spider.parse())：获得 response 后的回调函数
    • method(default='GET)：请求方法
    • meta(dict)：向回调函数传递参数的字典，有内置属性，也可自定义属性。由response.meta.get(<键>)中获得。

  • parse(response)

​ 通常作为 response 的回调函数，完成对当前页面的解析及数据提取(yield Item) 或 下一url的爬取跟进(yield Request)

拓展

IP与域名

  • IP地址：

​ IP地址是IP协议提供的一种统一的地址格式，用来标识每台网络中的主机，具有唯一性。

​ IP地址是32位二进制数，使用点分十进制表示，即每八位二进制数换算为十进制数，用点分割。

​ IP地址为了便于寻址被分割成网络号和主机号两部分，通过子网掩码区分网络号和主机号。

  • 子网掩码 (mask)：

    1表示网络位，0表示主机位，使用点分十进制表示。

  • 域名的意义：

​ 互联网通过IP地址建立访问，为了增强IP地址的可读性，通过网域名称系统 (DNS，Domain Name System) 将域名 (Domain) 和IP地 址相互映射.

  • 域名的结构：

    域名从右至左被点号分割为表示特定信息的不同部分——

    • 顶级域名 (Top-Level Domain, TLD)

      如最通用的顶级域名 (.com, .org, .net)，地区的顶级域名 (us, .fr, .sh)，政府部门 (.gov)，教育或研究机构 (.edu)。

    • 二级域名 (Secondary-Level Domain, SLD)

      处于顶级域名左边的域名为二级域名，二级域名左边为三级域名，以此类推。

      每级域名是其上级域名的子域名 (Subdomain)。
